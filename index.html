Native Web Components

Web components are made up of 4 technologies:

  1. Custom Elements
  2. HTML Imports
  3. HTML Templates
  4. Shadow DOM

Because all of these technologies are not supported fully in all browsers, polyfils must be used.

This page has been tested in Chrome, Firefox, Safari, and Opera. Behavior in other browsers is unpredictable, so please view using a supported browser

My web component: <alert-modal>

The web component I designed is the <alert-modal> component. As the name suggest, it is meant as a replacement for the built in JavaScript alert(). To start using this component, you must first import it (and the polyfill) to the page like so:

<script src="./webcomponents-lite.js"></script>
<link rel="import" href="./alert-modal.html">

Then just insert the alert-modal tag in the page's HTML. By placing text between the tags, the user can create their own alert message:

<alert-modal>Custom Alert Goes Here</alert-modal>

The alert is shown by using using the "visible" property. In this case, JavaScript is used to add the "visible" property to the element when the blue button is clicked below. To close the alert, press the red "Dismiss" button on the alert.

<alert-modal visible>Custom Alert Goes Here</alert-modal>
